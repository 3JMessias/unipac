Motivação de uso

  • Permitir o reaproveitamento de código já construído.
  • Evitar que um trecho de código que seja repetido várias vezes dentro de um mesmo programa.
  • Permitir a alteração de um trecho de código de uma forma mais rápida.
  • Melhorar o entendimento do código.
  • Criar uma divisão lógica do programa.
